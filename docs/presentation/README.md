# Personal Assistant - Presentation

Презентація командного проекту "Персональний помічник" для курсу Python Programming: Foundations and Best Practices.

## Технології

Презентація створена за допомогою [Slidev](https://sli.dev/) - сучасного фреймворку для створення презентацій на базі Markdown.

## Запуск презентації

### Встановлення залежностей

```bash
cd docs/presentation
bun install
```

### Запуск у режимі розробки

```bash
bun run dev
```

Презентація відкривається автоматично в браузері за адресою `http://localhost:3030/`

### Експорт у різні формати

**Експорт у PDF:**
```bash
bun run export
```

**Експорт у PPTX:**
```bash
slidev export --format pptx
```

**Експорт у PNG знімки:**
```bash
slidev export --format png
```

## Команди Slidev

Переміщення між слайдами:
- `Space`, `→`, `↓` - наступний слайд
- `Shift+Space`, `←`, `↑` - попередній слайд

Режими презентації:
- `f` - повноекранний режим (Presentation Mode)
- `c` - інструкція для спікера (Speaker Notes)
- `e` - відкрити редактор слайда

Експорт та збереження:
- `s` - зберегти слайд як зображення
- `d` - завантажити PDF
